Documentation

Splunk information

The data ingested needs to have the following Splunk specifications:

index=product
sourcetype=store_review

This app contains the following dashboards:

  • Rating Analysis - ratings over time and by location only
  • Rating and Sentiment Analysis - as above with additional sentiment analysis and world cloud
  • App Review Comment search - search for comments by country, rating and keyword

Data format

The data format should be as follows or contain the following fields:

{
"t": "06/Apr/2017:12:11:10",
"title": "This is a title in the original language",
"content": "This is the review in the orginal language! ",
"title_en": "This is the title in English",
"content_en": "This is the review in English",
"id": "1234abcd5678efgh",
"rating": 5,
"version": "1.1.1",
"device": "MyDevice",
"os_version": 1.1,
"author": "John Smith",
"country": "GB",
"store": "Android",
"app_name": "MyApp"
}


In each dashboard, choose the store in the drop down menu. If none appear then your data is not ingested properly.

Requirements

This app has only been tested for Splunk v6.5 and above. The rating analysis requires the Machine Learning (ML) toolkit. Some of the dashboards require the Sentiment Analysis App and the Wordcloud App, however, a simplified dashboard not using either exists too.
